# Figma

Pelagos styles can be used in Figma by loading them with the
[Tokens Studio plugin](https://www.figma.com/community/plugin/843461159747178978/Figma-Tokens).

Please follow the [URL synchronization documentation](https://docs.tokens.studio/sync/url)
using the name "Pelagos" and the URL "{new URL('/figma-tokens.json', location.href).href}".

To get theme tokens to work properly select only one of "white", "g100", "cg00", or "yg100" at a time.
