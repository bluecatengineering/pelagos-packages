# useSlidingPanel

Adds a "slide" animation to the specified element.

## Parameters

<table>
	<thead>
		<tr>
			<th>Param</th>
			<th>Type</th>
			<th>Description</th>
		</tr>
	</thead>
	<tbody>
		<tr>
			<td>id</td>
			<td>`string`</td>
			<td>the ID of the element to animate.</td>
		</tr>
		<tr>
			<td>onFinish</td>
			<td>`function`</td>
			<td>invoked when the animation finishes.</td>
		</tr>
	</tbody>
</table>

## Returns

`function` - function to use as event handler.

## Example

```js
import {useCallback} from 'react';
import {useSlidingPanel} from '@bluecateng/pelagos';

const Example = () => {
	const handleFinish = useCallback(() => {
		// handle finish
	});
	const handleCloseClick = useSlidingPanel('example', handleFinish);
	return (
		<div id="example">
			<button onClick={handleCloseClick}>Close</button>...
		</div>
	);
};
```
