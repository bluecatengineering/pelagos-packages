# useSelectPositioner

Returns an effect which positions the pop-up for a select box.

## Parameters

<table>
<thead><tr><th>Param</th><th>Type</th><th>Description</th></tr></thead>
<tbody>
<tr><td>open</td><td>`boolean`</td><td>whether the pop-up is open.</td></tr>
<tr><td>boxRef</td><td>`MutableRefObject.<Element>`</td><td>React ref to the select box.</td></tr>
<tr><td>popUpRef</td><td>`MutableRefObject.<Element>`</td><td>React ref to the pop-up box.</td></tr>
</tbody>
</table>

## Example

```js
import {useRef, useState} from 'react';
import {useSelectPositioner} from '@bluecateng/pelagos';

const Example = () => {
	const boxRef = useRef(null);
	const popUpRef = useRef(null);
	const [open, setOpen] = useState(false);
	useSelectPositioner(open, boxRef, popUpRef);
};
```
