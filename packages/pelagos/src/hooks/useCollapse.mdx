# useCollapse

Returns a React ref which collapses the element. The element must have `overflow: hidden`.

			<td>whether the component is expanded.</td>
		</tr>
	</tbody>
</table>

## Returns

`function` - React ref.

## Example

```js
import {useCollapse} from '@bluecateng/pelagos';

const Example = ({open}) => (
	<div style={{overflow: hidden}} ref={useCollapse(open)}>
		...
	</div>
);
```
